Reconciliation

This screen is used to view and reconcile positions of a trade day. Main functions of the screen are -
  1. Select date and the entity(PB or Admin) for reconciliation
  2. View all the positions for the selected criteria â€“ Unmatched, Matched and Matched with differences in quantities
  3. Enter and update comments for any position
  4. Map unmatched positions
  5. Unmap matched positions

Select Reconciliation criteria

  1. Positions will already be displayed according to a default criteria but it can be changed by clicking the filter icon
  2. Select the date and the entity along with the account for reconciliation
  3. Click apply
  4. The selected criteria will be displayed at the top of the page and the respective positions will be displayed in grids. The filter panel will close.

Enter position comments

  1. Click on the Comments column for any position
  2. Enter comments text in the input field
  3. Press enter, tab or click out of the column
  4. The entered comment will be saved

Map positions

  1. For any position In the Unmatched grid, click on any column apart from the Comments column
  2. The mapping panel will open up showing the references for the selected positions that can be mapped
  3. Select a reference by clicking on its radio button
  4. Select a security from the dropdown to map this position to that security
  5. Optionally, click on one of the suggestions to select that security for mapping
  6. Click Map
  7. The selected position will be mapped to the selected security and will now show up in the Matched grid

Unmap positions

  1. For any position In the Differences or Matched grid, click on any column apart from the Comments column
  2. The unmapping panel will open up showing the security and its reference to which the selected position is matched
  3. Click on Unmap
  4. The selected position will be unmatched and will now show up in the Unmatched grid
