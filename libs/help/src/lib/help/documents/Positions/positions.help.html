Positions

This screen is used to view the portfolio. A user can slice and dice the data by bucketing/grouping the portfolio in different categories or by applying various filters to the dataset. User can also drill down a holding and see further details about the position including its trade history.

Grouping the portfolio

Drag a field, with which you want to group the portfolio, to the blank row above.

Applying filters

  1. Click on the funnel icon
  2. Add filter
  3. Click on apply

Saving user settings

  1. Click on the cog-icon
  2. Add/Remove fields from available items
  3. Apply filter
  4. Click on file-icon to save

Position Details

This feature helps finding out more detail about a holding including
  1. Break of the position into various accounts at different custodians.
  2. Most recent price and the record date of the said price
  3. P&L associated with the said security for each account

Trade History for a position

This feature helps finding out the trades behind a particular holding
