Email Campaigns

Start an Email Campaign

  1. Navigate to Email Campaigns
  2. Select a distribution list
  3. Toggle contacts selection
  4. Click Next
  5. Edit the email content
  6. Add attachment
  7. Click Send Email
  8. The campaign will start and Pinnakl will start delivering emails

View Email Campaign Statistics

  1. Navigate to Email Statistics
  2. Adjust the search parameters if needed
  3. Click on your campaign if it's not already selected
  4. Click on a category to view the contacts

Start an Email Campaign with Tracking Links

  1. Start an Email Campaign
  2. Edit the email subject and body
  3. Copy the URL of the link you want to track
  4. Paste it in the URL textbox
  5. Enter description and the points you want to associate with the visitation of this link
  6. Click 'Track'
  7. Select the text to which you want to add the link
  8. Click Insert -> Link
  9. Paste the same URL here and click OK
  10. To upload your own file and make its tracking link, click 'Choose File' and select a file
  11. Click 'Upload' to upload the file and generate its URL
  12. Select the URL and copy it
  13. Paste the URL, enter description, points and click 'Track'
  14. Select the text in the email body and create the link to your file
  15. Click 'Send Email' to start the campaign

View Email Campaign Statistics for Tracking Links

  1. Navigate to 'Email Statistics'
  2. Click on the 'Opened' category
  3. Here you can view the total points scored by a contact
  4. Click on the 'V' expand icon to view the tracking links and points distribution
