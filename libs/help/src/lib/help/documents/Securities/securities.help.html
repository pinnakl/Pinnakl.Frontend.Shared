Securities

This screen is used to setup securities in the Pinnakl system. Only once a security is available in the Pinnakl system, can the end user book a trade for it in OMS. Main functions of this screen are -
  1. Add new securities
  2. Modify existing securities
  3. Add new organizations
  4. Modify existing organizations

Add new Security

  1. Select “Add New”
  2. Select “Add automatically” if it’s a listed security. Else select Add Manually.
  3. Select Identifier Type
  4. Enter Identifier
  5. Click Add.
  6. Pinnakl will setup the security via IDC and the user will be directed to the security screen that shows the details. At that point, user can change the details or move out of the screen

Modify existing security

This screen is used to modify the setup in case a security attribute changes including accruing status, coupon etc.
  1. Select “Edit Security”
  2. Select security
  3. Make changes
  4. Hit Save.

Add Organization

This screen is used to add the organizations associated with securities
  1. Go to the securities screen and under the Manage Organization icon, click the Add/Edit button
  2. Enter Organization details
  3. Click Save

Edit Organization

This screen is used to modify the details of an existing organization
  1. Go to the Manage Organizations screen
  2. Select an organization to edit using the search box at the top of the screen
  3. Modify organization details using the input fields
  4. Click Save
